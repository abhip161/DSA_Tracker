# DSA Tracker Web App

Full-stack DSA tracker with authentication, problem CRUD, dashboard stats, and filtering.

## Tech Stack
- Frontend: React (Vite), Tailwind CSS, Axios
- Backend: Node.js, Express, MongoDB (Mongoose), JWT

## Project Structure
- `backend/` Express API
- `frontend/` React app

## Quick Start

### Backend
1. `cd backend`
2. `copy .env.example .env` (Windows) or `cp .env.example .env` (Linux/macOS)
3. Update `.env` values
4. `npm install`
5. `npm run dev`

### Frontend
1. `cd frontend`
2. `copy .env.example .env` (Windows) or `cp .env.example .env` (Linux/macOS)
3. `npm install`
4. `npm run dev`

Frontend runs on Vite default port (usually 5173) and backend on 5000.
